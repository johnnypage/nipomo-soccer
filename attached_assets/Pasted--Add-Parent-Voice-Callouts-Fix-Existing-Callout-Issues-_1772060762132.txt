## Part 1: New Callout Type — Parent Voice

These are real quotes from the end-of-season survey. They need their own visual treatment that's clearly distinct from the existing pull quotes (green left border, italic).

### Design spec

- **Opening quotation mark**: Large decorative `"` in green (#2E7D32), roughly 36–48px, positioned top-left of the quote as a visual accent — not inline with the text
- **Quote text**: Regular weight (not italic — that's for board voice pull quotes), slightly larger than body text (~18px), dark foreground color
- **Attribution**: Below the quote, smaller text, muted gray: `— Nipomo soccer parent, end-of-season survey`
- **Background**: White or very slightly warm white (`#FDFCFA` or similar) — NOT the cream card background used by pull quotes. This subtle difference separates the two quote types visually.
- **Border**: None. The decorative quotation mark does the work. No left border (that's the pull quote treatment).
- **Spacing**: Generous padding inside, comfortable margins above and below

The goal: when someone scrolls past a pull quote and then a parent quote, they should immediately feel like different voices are speaking — the board explaining, and then a parent confirming.

## Part 2: Fix Existing Callout Issues

### Fix 1: AYSO pull quote — make it visually distinct

The pull quote that reads *"Many issues can only be solved by your local AYSO" — from AYSO's own support pages* currently looks identical to the other two pull quotes (green left border, italic).

This one should look different because it's not the board's voice — it's AYSO's own words being used as evidence. Change its styling:

- **Left border color**: Change from green to a muted gray or the maroon (#8B1D24) — this is the opponent's words, not ours
- **Add a subtle background tint** that's slightly different from the other pull quotes (a very light gray, or a very light warm tone)
- Keep the italic, but this visual distinction separates "our voice" pull quotes from "their words" pull quotes

### Fix 2: "200+ / Referees" stat callout

The current two-column stat callout has "200+" on the left and "Referees" on the right. The problem is one side is a number and the other is just a word — it reads unbalanced compared to the 25/5/1 block which works because all three columns are numbers.

Replace with a single centered stat:

**200+** volunteers needed to run the season — and referees were the hardest to find

Style it as a single-line stat with the number big and bold, and the descriptor text below it in one line. No divider, no two-column layout.

## Design Notes

- Parent voice callouts should animate in on scroll the same way other callouts do
- Don't overdo it — 5 parent quotes is the max. The article shouldn't feel like a testimonials page. These are surgical placements that validate a specific argument being made in the surrounding text.
- On mobile, the decorative quotation mark should scale down but still be visible
- Make sure no quote text is duplicated in the body paragraphs around it
