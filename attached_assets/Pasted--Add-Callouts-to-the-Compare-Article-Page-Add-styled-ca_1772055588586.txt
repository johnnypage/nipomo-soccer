# Add Callouts to the Compare Article Page

Add styled callout elements throughout the article on `/about/compare` to break up large text blocks and highlight key stats, quotes, and takeaways. These should feel native to the existing design system.

---

## Callout Styles Needed (3 types)

### 1. Pull Quote
A centered, visually distinct quote block. Use a left border accent (green `#2E7D32`), slightly larger italic text, generous padding, and the card background (`#F8F5F1`). No quotation marks needed, just the styling should signal "this is important."

### 2. Stat Callout
A compact, high-contrast box for key numbers. Dark background (`#0D0D0D`) with cream text (`#F3ECE2`). Numbers should be large and bold. Can include 1-3 stats side by side. Rounded corners, centered.

### 3. Key Point Callout
A subtle highlighted box for important takeaways. Use a light green-tinted background with a thin green left border. Body-sized text, bold. Think "highlighted paragraph" not "alert box."

## Design Notes

- Callouts should have consistent spacing (margin top and bottom) so they breathe between paragraphs
- On mobile, stat callouts with multiple numbers should stack vertically
- Pull quotes should never be wider than the article text column
- Animate callouts with a subtle fade-in on scroll if the page already has scroll animations
- Make sure the callout text is NOT duplicated in the body text. If a callout pulls a sentence from a paragraph, either remove that sentence from the paragraph or rephrase the paragraph so it flows without the extracted line.
